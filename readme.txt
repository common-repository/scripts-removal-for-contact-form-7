== Description ==

By default Contact form 7 CSS and JS includes on all the pages, which will add extra unwanted load on the pages.

This plugin eliminate above limitation. And remove the unwanted scripts from all the pages.

== Installation ==

1. Go to plugins->Add New. Click on Upload Plugin.
2) Browse the downloaded zip file And click install now.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You are Done.
